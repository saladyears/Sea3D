Sea3D Keyboard Shortcuts

NOTE: At any time, pressing F1 brings up the help system.

Keyboard shortcuts in the main Sea3D window.

Ctrl+N

Launches the What do you want to do? window so you can start a new Sea3D game.

Ctrl+O

Opens the incomplete game database.

Alt+F4

Shuts down Sea3D completely.

Keyboard shortcuts in the main OpenGL window.

Ctrl+S

Saves the current Sea3D game to the incomplete game database (if possible).

Ctrl+F4

Closes the current Sea3D game but does not quit Sea3D.

Ctrl+A

Saves the current camera angle as the default camera angle.

Ctrl+M

Clears the current camera angle back to Sea3D's original angle.

Alt+F4

Shuts down Sea3D completely.

Ctrl+Z

Undoes the last action (if possible).

Keyboard shortcuts during game play.

Arrow Keys

Rotate the map.

Ctrl + Up and Down Arrow Keys

Zoom the map.

Shift + Arrow Keys

Pan the map.

Ctrl+R

Roll the dice on your turn.

spacebar

Continue after you've rolled.

Ctrl+P

Play a card from your hand.

Ctrl+B

Trade with the bank.

Ctrl+T

Trade with others.

Ctrl+E

End your turn.
